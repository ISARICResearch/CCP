This prospective observational study used data collected in [SITE/LOCATION] through the ISARIC/WHO Clinical Characterisation Protocol for Severe Emerging Infections 1,2. Data capture was conducted using a case report form (CRF) based on ISARIC ARC 3, a structured, machine-readable library of standardised questions designed to support rapid and harmonised data collection across outbreaks. The CRF used for characterization of [DISEASE] during this outbreak was adapted for this specific study context using ISARIC BRIDGE4, a web-based platform that facilitates CRF design and customisation.

Data analysis were generated performed in ISARIC VERTEX4, a web-based application that allows analisis of ISARIC ARC-formatted data and generates tables and visualizations to support key research questions. VERTEX is part of the ISARIC Clinical Epidemiology Platform, developed to standardize and streamline data collection and analysis in outbreak-related clinical research. A public implementation of VERTEX for this study is available at: [Public vertex project webpage adress]
